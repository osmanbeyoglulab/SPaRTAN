Affinity regression basic simulation code.
run_helloworld.m shows a basic simulation of generating a W matrix and recovering it.
run_simulation.m shows a more comprehensive simulation and produces the simulation results in the supplementary note figure 9. 

To run the code please install SLEP:
http://www.public.asu.edu/~jye02/Software/SLEP/

Also, add the folder packages to the matlab path.

A full package is available at the following public git repository:
https://bitbucket.org/leslielab/affreg
