SPaRTAN basic MATLAB demo.

To run the code please install SLEP:
http://www.public.asu.edu/~jye02/Software/SLEP/

Also, add the folder packages to the matlab path.

We include affinity regression package. A full affinity regression package is available at the following 
public git repository:
https://bitbucket.org/leslielab/affreg
